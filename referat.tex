\bigskip

\noindent ОПТИМАЛЬНОЕ УПРАВЛЕНИЕ, ПРОГНОЗИРУЮЩАЯ МОДЕЛЬ, ТЕХНОЛОГИЧЕСКИЙ ПОСЛЕДОВАТЕЛЬ, ПРОГНОЗИРУЮЩАЯ ЗАДАЧА ОПТИМАЛЬНОГО УПРАВЛЕНИЯ, EMPC, АЛГОРИТМ

\bigskip

Объектом исследования является процесс экономического развития страны технологического последователя, заимствующего технологии страны-лидера.

Цель работы --- проанализировать решение задачи оптимального экономического роста страны-последователя при различных параметрах с использованием трёх подходов построения EMPC.

Методы решения --- для решения и исследования поставленной задачи используются: решение с помощью принципа максимума и методы управления по прогнозирующей модели для экономических задач.

Результатами работы являются --- выбор наиболее эффективного метода для решения задачи.
