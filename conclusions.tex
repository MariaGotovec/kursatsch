\chapter*{Заключение}
\addcontentsline{toc}{chapter}{ЗАКЛЮЧЕНИЕ}

В данной дипломной работе приведены: основные результаты теории оптимального управления, а также основные результаты качественной теории и наиболее часто применяемые на практике численные методы решения задач оптимального управления\cite{Diehl}.

Описываются: основные положения теории управления по прогнозирующей модели --- МРС (Model Predictive control)\cite{Fontes, Grune}, основной алгоритм построения МРС-регулятора, некоторые подходы к обеспечению устойчивости системы, замкнутой построенной обратной связью и экономический МРС (ЕМРС), который является модификацией MPC.

Рассмотрена задача оптимального роста технологического последователя. Для данной задачи была построена математическая модель роста технологического последователя, заимствующего технологии страны-лидера, приводиться согласно идеям работы \cite{Kryazhimski}. Для рассматриваемой модели сформулирована задача оптимального управления, которая решается с использованием Принципа максимума. Также эта задача была решена численно.

Приведено программное решение задачи оптимального роста технологического последователя с помощью методов управления по прогнозирующей модели для экономических задач (EMPC). Алгоритмы решения задачи были реализованы в MatLab с помощью CasADi.

Проведены численные эксперименты при одинаковых параметрах для трёх подходов: базового EMPC, EMPC с терминальным ограничением-равенством и EMPC с терминальной стоимостью, в которых было продемонстрировано превосходство третьего подхода. Данный подход упрощает задачу оптимального управления с точки зрения численного решения, поскольку задачи без жестких ограничений-равенств проще с вычислительной точки зрения.
