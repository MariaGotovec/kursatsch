\chapter*{Заключение}
\addcontentsline{toc}{chapter}{ЗАКЛЮЧЕНИЕ}

В обзорной части данной дипломной работы (главы 1 и 2) приведены основные результаты теории оптимального управления,  наиболее часто применяемые на практике численные методы решения задач оптимального управления\cite{Diehl}, даны основные положения теории управления по прогнозирующей модели \cite{Fontes, Grune}, алгоритм построения МРС-регулятора, некоторые подходы к обеспечению устойчивости системы, замкнутой построенной обратной связью, и экономический МРС (ЕМРС), который является модификацией MPC.

В основной части работы (главы 3 и 4) методами  оптимального управления и методами EMPC исследуется  задача оптимального роста технологического последователя. Для данной задачи построена математическая модель роста технологического последователя, заимствующего технологии страны-лидера. Модель приводится согласно идеям работы \cite{Kryazhimski}. Для рассматриваемой модели роста сформулирована задача оптимального управления, которая решается с использованием принципа максимума. 

Существует множество экономических задач оптимального управления с бесконечным горизонтом, решение которых представляет практический интерес, однако не может быть получено аналитически, с использованием результатов \cite{Kryazhimski}. На примере задачи роста технологического последователя в  работе предлагается подход к решению, основанный на использовании методов управления по прогнозирующей модели, в частности методов ЕМРС.

В дипломной работе сравниваются три варианта ЕМРС: 1. неограниченный ЕМРС, 2. ЕМРС с терминальным ограничением типа равенства, 3. ЕМРС с терминальной стоимостью. Исследуются параметры, позволяющие получить решение, асимптотически приближающее магистральное решение, установленное аналитически с помощью принципа максимума.

Программное решение прогнозирующих задач оптимального роста технологического последователя для всех трех подходов было построено с помощью прямого метода Direct Single Shooting, который реализован в MatLab с помощью пакета CasADi.

В результате численных экспериментов продемонстрировано превосходство третьего подхода. Для него также предложен простой вариант выбора линейной терминальной стоимости. ЕМРС с терминальной стоимостью упрощает задачу оптимального управления с точки зрения численного решения, поскольку задачи без жестких ограничений-равенств проще с вычислительной точки зрения.
